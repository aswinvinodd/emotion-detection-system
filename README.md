# Emotion Detection & Sentiment Analysis System

An AI-powered web application that detects human emotions and sentiment from textual feedback using Natural Language Processing (NLP) and Streamlit.

---

## 📌 Project Overview

This system analyzes user text input and identifies:

- Emotion (Joy, Sadness, Anger, Fear, Disgust, Surprise)
- Overall Sentiment (Positive / Negative / Neutral)
- Confidence Score
- Sarcasm Detection
- Analytics Dashboard

The application provides visual insights and stores analysis history for further evaluation.

---

## 🚀 Features

✅ Real-time Emotion Detection  
✅ Sentiment Classification  
✅ Confidence Gauge Meter  
✅ Batch Text Analysis  
✅ History Storage & Search  
✅ Sentiment-Based Sorting  
✅ Analytics Dashboard  
✅ PDF Report Download  
✅ Dark / Light Mode Toggle

## ⚙️ Installation & Run

### 1️⃣ Clone Repository

```
git clone https://github.com/aswinvinodd/emotion-detection-system.git
cd emotion-detection-system
```

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 3️⃣ Run Application

```
streamlit run src/app.py
```
